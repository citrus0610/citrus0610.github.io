LeetCode 994. Rotting Oranges - 解題紀錄

題目

連結
難度：Medium

給定 m x n 的矩陣
裡面會有 3 種值
0：空的格子
1：新鮮的水果
2：腐爛的水果

每一分鐘，腐爛水果的上下左右新鮮水果都會變為腐爛的
求至少要經過幾分鐘，水果才會全部變成腐爛的
如果無法所有水果都變成腐爛的，則回傳 -1

解法：BFS

首先遍歷一次矩陣，將腐爛水果的格子放進 queue 裡
並且記錄新鮮水果的數量
遍歷完矩陣後，如果新鮮水果的數量為 0
代表沒有新鮮水果，已完成題目條件，回傳 0

接著開始 BFS
每次判斷新的格子是否為新鮮水果
如果是的話才加進 queue 裡，並且將新的那格變為腐爛的
新鮮水果的數量也要減 1

最後判斷新鮮水果數量是否為 0
如果不是，代表有到不了的新鮮水果格子，回傳 -1
如果是，則回傳 BFS 的層數減 1

Runtime：1ms - Beats 89.99%
