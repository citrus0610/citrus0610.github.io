LeetCode 543. Diameter of Binary Tree - 解題紀錄

連結
難度：Easy

求 binary tree 裡，2 個 node 之間的最長距離

解法：DFS

遞迴，每次回傳左子樹和右子樹之中的最大深度
並且額外紀錄 左子樹深度 + 右子樹深度 的最大值，即為答案
